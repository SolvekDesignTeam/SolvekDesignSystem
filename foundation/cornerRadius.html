Corner radius

일관된 둥근 모서리 스타일을 통해 제품의 전체적인 톤앤매너를 유지하고, 컴포넌트 간 통일성 있는 사용자 경험을 제공합니다.

Corner radius Naming Convention(둥근 모서리 네이밍 규칙 설명)

솔브케이 디자인 시스템에서는 둥근 모서리 계층을 일관성 있고 직관적으로 정의하기 위해 다음과 같은 네이밍 규칙을 사용합니다.

Radius-XS-2

CSS border-radius:2px를 의미

Corner radius Naming Convention(둥근 모서리 네이밍 규칙 설명)

radius-full은 단순히 모양을 위한 것이 아니라 기능적으로도 다른 버튼들과 구분되는 중요한 행동에 사용하세요.
예를 들어 추가,삭제,수정 등 해당 페이지에 기본으로 수행하는 기능이 아닌 특수 기능 트리거 같은 버튼에는 사용자가 직관적으로 차이를 느낄 수 있도록 radius-full을 적용할 수 있습니다.

corner radius 토큰 정의

Token Name

Pixel Value

사용 용도
radius-xs-2 2px 테이블 안에 들어가는 작고 정형화된 버튼, 태그, 뱃지 등
radius-sm-4 4px 소형 버튼, 입력 필드, 작은 카드 컴포넌트 등
radius-md-8 8px 일반 버튼, 입력창, 카드 등 대부분의 기본 컴포넌트에 사용
radius-lg-16 16px 모달, 팝업, 모바일 바텀시트 등 넓은 면적의 컴포넌트에 사용
radius-full 100% 완전한 원형 형태가 필요하거나, 특수한 기능을 시각적으로 강조할 때 사용.
